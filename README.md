# sapigo

Simple API made in Go. Toy project to test how [gin](https://gin-gonic.com/) works.

# API

- v1/student
- v1/student/:id

# Usage

Clone the repository
```
git clone https://github.com/SrVariable/sapigo
```

Run the program
```
go run ./cmd/sapigo
```
